# Professional Portfolio

This repository contains the source code for the personal portfolio website of **Gregorius (陈春天) Karisma**, an Analytics Professional specializing in Semiconductor Fabrication and Data Analytics.

## Live Demo
## Features
* **Bilingual Support:** Full content available in **English** and **Mandarin Chinese (中文)**, with a persistent language toggle that saves user preference.
* **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
* **Dark Mode UI:** A professional, high-contrast dark theme suitable for tech and engineering portfolios.
* **Zero Dependencies:** Built with pure HTML5, CSS3, and Vanilla JavaScript (no frameworks or libraries required).
